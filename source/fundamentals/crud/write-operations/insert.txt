Overview
--------

In this guide, you can learn how to insert documents into your MongoDB
collection. To insert a document, use insert operations.

Insert operations consist of the :java-sync-api:`insertOne()
<com/mongodb/client/MongoCollection.html#insertOne(TDocument)>` 
and :java-sync-api:`insertMany()
<com/mongodb/client/MongoCollection.html#insertMany>`
methods. For both methods, the driver automatically generates a unique
``_id`` for documents unless the document specifies an ``_id``. 

Insert a Document
-----------------

Use the ``insertOne()`` method to insert a single document into a
collection. This method returns an :java-docs:`InsertOneResult
<apidocs/mongodb-driver-core/com/mongodb/client/result/InsertOneResult.html>`
instance which tells us the ``insertedId`` of the document inserted.

collection. This method returns an :java-docs:`InsertManyResult
